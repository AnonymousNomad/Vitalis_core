# Ferrell Synthetic Intelligence (FSI)
A self-evolving, Hebbian-driven cognitive kernel for autonomous agency.

## Features
- **Hebbian RNN Core:** Dynamic weight adaptation without backpropagation.
- **Bayesian Confidence Gate:** Prevents hallucination/untrusted actions via threshold-based logic.
- **Persistent State:** Brain states serialized for continued evolution.

## Installation
```bash
pip install .
```

## Usage
Launch the intelligence kernel:
```bash
fsi-chat
```

## Mission
FSI is designed for operational autonomy, shifting the paradigm from static LLMs to living, evolving agents.
